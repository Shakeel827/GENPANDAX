## About

GenPandax AI Labs is a cutting-edge digital agency based in India, partnering with startups and enterprises worldwide. We craft immersive web experiences and intelligent business systems powered by modern technology and AI.

## What We Do

- **Web Development** — Custom, high-performance websites built with React, Next.js, and TypeScript
- **Business Systems** — CRM, inventory management, analytics dashboards built to scale
- **UI/UX Design** — Pixel-perfect interfaces with immersive animations and interactions
- **Performance & SEO** — Core Web Vitals optimization and organic growth strategies

```bash
# Run development server

# Build for production
npm run build
```
